# ShrekOS
A browser-based desktop OS with physically simulated windows for Shrek enthusiasts and anyone who likes browsing.

This project is a web based operating system built with HTML, CSS, and JavaScript, using DOM manipulation to create interactive apps, draggable windows, and a desktop-style interface.

- ShrekOS includes three apps that, upon being clicked, open into simulated, draggable windows
    - **Characters**: An app including information about various film characters
    - **Notes**: Includes journal type notes from the user
    - **Calculator**: Uses a double number input and displays arithmetic solutions
- The welcome page can be opened by clicking "Shrek OS" in the top left
- The date and time are displayed in the top right through Javascript's Date object and DOM manipulation
